#  Electricity Bill Calculator

A web app to calculate and generate electricity bills instantly.
Built by **Himon Protim Medhi**

## 🔗 Live App
[Click here to open the app](https://himonp.github.io/Electricty-BILL-calculator)

##  Features
- Enter previous and current meter readings to calculate units consumed
- Custom rate per unit — set your own ₹ rate, not fixed at ₹14
- Clean bill generated instantly with customer name and dates
- **Signature pad** — draw an authorized signature directly on the bill
- **Share as Image** — exports the bill as a PNG and shares directly via WhatsApp
- Works on both mobile and desktop

##  Built With
- HTML, CSS, JavaScript
- html2canvas (for bill image export)
